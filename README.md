# 은우의모험 v0.2.0

## 단계 구성
- 1단계: 기존 아재개그 50문제 / MC1
- 2단계: 상식 퀴즈 50문제 / MC2
- 3단계: 아재개그 50문제 / MC3
- 각 단계에서 정답 5개를 맞히면 다음 단계 버튼이 나타납니다.
- 단계가 바뀌면 점수와 해당 단계의 출제 기록이 초기화됩니다.
- 한 단계 안에서는 이미 출제한 문제가 중복되지 않습니다.
- 3단계 완료 후에는 처음부터 버튼으로 1단계로 돌아갑니다.

## UI 변경
- 우측 상단 제목을 삭제하고 다시 시작 버튼을 추가했습니다.
- 다시 시작 버튼은 언제든 1단계, 점수 0점부터 시작합니다.

## MC 이미지
현재 업로드 파일에 `mc2_idle.png`, `mc3_idle.png`가 없어 기존 MC 이미지를 임시 복제했습니다.
새 이미지를 준비한 뒤 `assets/mc2_idle.png`, `assets/mc3_idle.png`에 덮어쓰면 코드 수정 없이 적용됩니다.
두 파일 모두 기존 MC와 동일하게 2프레임 가로 스프라이트 형식을 사용합니다.
